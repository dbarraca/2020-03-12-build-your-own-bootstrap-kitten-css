Explanation

Voila is a css framework with ready made components. You just need to include the css file. The fil found in the kitchen sink section below. You simply need to add the needed class to create the component. We like to keep things short and sweet. The classes are common class names used in other css frameworks with the vowels removed.Just like that you will be yelling out "Voila!" as you look upon the beautiful website you have created.
