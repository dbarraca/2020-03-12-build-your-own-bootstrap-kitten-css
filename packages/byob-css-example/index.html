Voila

Voila is a CSS framework that will help you quickly cook up your next project.

GitHub

How to use Voila CSS

Voila is a css framework with ready made components. You simply need to download the Voila css file and include the css file in your head tag. Then you can add the needed classes to your html to create each component.The available components can be found in the kitchen sink section below. Just like that you will be yelling out "Voila!" as you look upon the beautiful website you have created.

Also, include this css font file from Google fonts to have the Montserrat file family.
